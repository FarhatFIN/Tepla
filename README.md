# Tepla
Tepla — это современный мессенджер, который я сейчас разрабатываю. Проект построен на Next.js, TypeScript и real-time архитектуре. Уже реализованы авторизация, система чатов, онлайн-статусы, индикатор печатания и криптографический модуль. В будущем планируются мобильные приложения, звонки, медиа и новые функции для безопасного общения
